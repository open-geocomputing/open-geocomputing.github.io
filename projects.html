GEEex

GEEex is a Google Chrome extension for Google Earth Engine that allows direct import of dataset using standard ingestion files by simple drag and drop. This allows the ingestion of advance data set with separate band file, pyramid policy, band names, mask, etc. Furthermore, the extension allows direct search and import of datasets from Planet into GEE, through the GEE interface. This project was recently migrated in Open Geocomputing expecting some synergie developpement with OpenEEL.
