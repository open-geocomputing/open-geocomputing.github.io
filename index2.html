Our goal

The goal of the Open Geocomputing is to help the community by providing and hosting free and open project.

Our philosophy is to keep to have a very open philosophy😉, we have almost not restriction about the type of contribution hosted each request is studies by the community and can go from a single script to a complete data set, and pass naturally by software, toolbox and libraries (if you have other ideas, don't hesitate to surprise us 😉). The few restriction:

  1. License restriction; we require that contributions are free and open, available for the communities to build on. The use of MATLAB and other close software are allowed 😉 as far as the scripts, functions are open.
  2. Maintainability restriction; we request that the project is maintanable, we will try as hard as psoible to limit the number of zoomby projects. Therfore, we will request a mininim maintenability, and not only a contribution written for the author in the context of abscur publication.
  3. Documentation restriction: this is more a code of conduct than a restriction, we request that project that start to grow have a minimum of documentation. The goal is not to force people to write a complete guide about an obvious fucnction but to improuve usability.
